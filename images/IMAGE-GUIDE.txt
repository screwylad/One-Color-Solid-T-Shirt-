## üì∏ Image Guidelines

### Product Photos (Front & Back)
- **Aspect Ratio**: 3:4 (portrait orientation)
- **Recommended Size**: 600x800px minimum
- **Format**: JPG
- **Background**: Clean, solid color background (white or light gray preferred)
- **Consistency**: All photos should have similar lighting and styling

### Hero Banner
- **Aspect Ratio**: 16:9 or similar wide format
- **Recommended Size**: 1920x600px or larger
- **Format**: JPG
- **Content**: Lifestyle image, model wearing t-shirt, or brand visual
- **Note**: Text will be overlaid on this image, so choose one with good contrast

## ‚ö†Ô∏è Troubleshooting

**Images not showing?**
- Check that filenames match exactly (case-sensitive)
- Verify images are in the correct folder
- Make sure file extensions are `.jpg` (not `.jpeg`, `.png`, etc.)
- If using different formats, update the HTML accordingly

**Want to use different colors?**
- Update the product card in `index.html`
- Change the image filenames
- Update the product name text
